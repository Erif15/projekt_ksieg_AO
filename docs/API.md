# API – BookNest

API działa lokalnie pod adresem:
http://localhost:4000

## Endpointy

### GET /health
Sprawdza czy API działa.

### GET /books
Zwraca listę książek.

### GET /books/:id
Zwraca szczegóły książki.

### POST /books
Dodaje nową książkę.

### PUT /books/:id
Aktualizuje książkę (tytuł/autor/opis/ocena).

### DELETE /books/:id
Usuwa książkę.
